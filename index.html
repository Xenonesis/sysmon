Capabilities

Everything you need to see,
nothing you don't.

A focused set of tools for developers, sysadmins, and power users who need real data fast.

Real-time Dashboard

CPU, memory, GPU utilization with color-coded progress bars. Green-yellow-red thresholds adapt automatically.

Core

Performance Graphs

Live line charts tracking CPU, memory, and GPU over time. Spot anomalies and trends at a glance.

Process Manager

Sort by CPU, memory, or PID. Search, inspect, and kill rogue processes without leaving the app.

NVIDIA GPU Metrics

Dedicated NVML integration for GPU utilization, VRAM, and core temperature monitoring.

Per-Core CPU View

Individual core utilization breakdown. See exactly which cores are loaded and which are idle.

Smart Alerts

Configurable thresholds for CPU, memory, GPU temp and disk space. Get warned before things break.
